  1. Radix Sylva : exécuter les imports nécessaires (import_hydroquebec, import_vascan, etc.) puis éventuellement merge_organism_duplicates, rebuild_search_vectors côté Radix.
  2. Jardin bIOT : lancer sync_radixsylva (option full au premier tirage). Vérifier RADIX_SYLVA_API_URL / clé sync dans .env.
  3. Semences / inventaire : toujours via Importer semences (catalogue BIOT).
  4. Import PFAF depuis l’admin Django reste disponible pour transition ; à terme, privilégier Radix + sync.
